# Telecom Information System (Flask)

Web-based information system developed as a university course project (2025).

Built with Flask and SQLite.  
Implements role-based access control (guest, user, operator, admin) and full CRUD functionality.

## Features

- Authentication with password hashing
- Role-based access control
- Subscriber management
- Debt tracking
- Repair tracking
- Number change requests
- Built-in SQL reports
- Custom SQL query execution (academic purpose)

## Tech Stack

- Python
- Flask
- SQLite
- Jinja2

## Run locally

```bash
pip install -r requirements.txt
python app.py
```
Database file is created automatically on first run.
